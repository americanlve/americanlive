  • If you've opted to use an authorization code, select where you'd like to have the code sent and enter the code.
  • If asked, answer the personal challenge questions you chose when you created your user ID
  • When logging in from a trusted device (such as a home computer), you can check Save user ID to simplify future logins.
  • Visit the Security Center after you log in for more details about extra security.
Select Forgot user ID and enter the email address associated with that user ID. Your user ID will be sent to the email address your provided.

If you don't recognize the user ID on the login page, you can clear it by selecting the box and using the backspace or delete key. Then enter user ID in the box.

If you're using a trusted device (such as your home computer), select Save user ID to simplify future logins.

Make sure you've typed in your password correctly. Remember, it's case-sensitive.

  • Select Forgot password and enter the required information.
  • Confirm your identity, and an authorization code will be delivered to your phone.
  • Enter and confirm your new password.
  • Passwords must contain:
    • 8 to 20 characters
    • At least 1 upper-case letter, 1 lower-case letter and 1 number
    • No number or letter repeated more than 3 times consecutively
    • No spaces and use only these characters: @ # * ( ) + = { } / ? ~ ; , . - _

You may be asked to answer personal challenge questions during login if you:

  • Are logging in from a new device for the first time.
  • Have not previously selected Save user ID on this device.
  • Have recently cleared cookies or reset settings on this device.

Be sure to answer the questions exactly as you entered them when you first chose the questions. Any punctuationused during the setup process must be included in your responses to these questions.

For example, if you answered St. Mary's, you should enter St. Mary's, not StMarys.

If you have forgotten your answers, contact us.

If you can't log in and need to change your personal challenge questions, contact us.

If you're logged in and want to change your security questions, visit the Help & Support menu and selectSecurity Center. After you've confirmed your identity by entering your password, select Personal challenge questions to make changes.
